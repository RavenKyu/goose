goose
=====

개인용 CMS Goose입니다.
거위라는 프로그램의 사용 용도는 글이나 사진이 담긴 포스트를 관리하는 프로그램이라고 볼 수 있습니다. 대략적으로 작은 워드프레스로 생각해도 되겠습니다.
주요 포스트 데이터는 둥지(Nest)라는 장소에서 관리하고 그 데이터들은 다른 공간에서 불러서 사용할 수 있습니다.


##설치환경
Goose 프로그램을 설치할 수 있는 환경은 XE나 워드프레스를 설치할 수 있는 아래 조건을 충족하는 환경이라면 쉽게 설치할 수 있습니다.

* Apache 2.0
* PHP 5.3
* MYSQL 5.5

##Installation
1. 파일을 받아서 서버에 압축풉니다. `/www/goose/`
2. 웹브라우저에서 goose파일을 설치한곳을 주소로 적어서 페이지 이동합니다. `http://domainaddress.com/goose/`
3. **Install Goose**설치화면이 나옵니다.
4. db정보와 관리자 정보를 입력합니다. 관리자 정보 섹션에서 **이메일**과 **비밀번호**는 로그인에 사용됩니다.
5. 설치하기 버튼을 누르면  alert창에서 "Complete install"라는 메세지가 나오면서 설치완료됩니다.
6. 로그인화면으로 이동하는데 아까 적었던 관리자 이메일주소와 비밀번호를 입력합니다.
7. 이제 씁니다.
